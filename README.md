# Worldline Security Champions Program

Slides for the talk presented at **OWASP AppSec Days France 2025**,
published as a web presentation with [Reveal.js](https://revealjs.com).

**Live:** https://adaptivui.github.io/security-champions-slides/

## About
Building a Security Champions program in a highly regulated environment:
the innovations that worked, the strategic challenges, and practical
recommendations to launch your own program.

## Navigation
- Arrow keys or click to move between slides
- `F` fullscreen · `Esc` slide overview
- `S` speaker view (presenter notes)
- Add `#/<n>` to the URL to deep-link a specific slide

## Speaker notes
Presenter notes are available for every slide and open in a separate
**speaker window** with the `S` key (current slide, next slide, notes and
a timer). Notes are shown **only** in speaker mode — they never appear on
the main presentation view.

> Tip: allow pop-ups for the site so the speaker window can open.

## Run locally
Just open `index.html` in a browser, or serve the folder:

```bash
python3 -m http.server 8000
# then open http://localhost:8000
```

## Export to PDF
Open the presentation with the `?print-pdf` query string, then print to PDF
(margins: none, background graphics: on). Use a Chromium-based browser
(Chrome / Edge) for reliable output. Speaker notes are **not** included in
the PDF.

```
https://adaptivui.github.io/security-champions-slides/?print-pdf
```
